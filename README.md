# 🚀 projectmgmt Backend

Node.js + Express backend for a project management system.

## ⚠️ Status

🚧 Work in progress — Authentication completed, Project & Task features coming next.

## ✅ Features (Done)

* JWT Authentication (Access + Refresh Tokens)
* User Register & Login
* Email Verification
* Forgot & Reset Password
* Protected Routes

## 🚧 Coming Soon

* Project Management APIs
* Task & Subtask System
* Role-Based Access Control (RBAC)

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* JWT
* Nodemailer

## ▶️ Run Locally

```bash
npm install
npm run dev
```
